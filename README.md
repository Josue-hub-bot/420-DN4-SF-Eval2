## Description du projet

Ce projet a pour but de faciliter le déploiement et la gestion d'une application web utilisant Ansible et Docker. L'application est déployée à l'aide de Docker Compose, avec une configuration Nginx pour servir des fichiers PHP. Le projet inclut des scripts et des fichiers de configuration pour automatiser l'installation de Docker et la gestion des environnements de production.

Les principaux objectifs du projet incluent :

- Automatisation du déploiement d'une application web via Ansible.
- Utilisation de Docker et Docker Compose pour la gestion des conteneurs.
- Configuration de Nginx pour servir un site web dynamique.
- Sécurisation de l'accès aux variables sensibles à travers Ansible.
